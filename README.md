# TripleTen web_project_expenses
## Project description
In this project (part of the Tripleten bootcamp) I worked in a budget planning web page where the user can interact by setting a budget, adding and removing expenses. The web page will show some information like the total expenses, average expense, the available money, the category with the largest total expense and some statistics by category.

Here I mainly practiced basic JavaScript by creating variables and functions for the page to work, show the correct information and make it interactive.
